# Product 1

Local self-hosted L2 Reborn auto-vote app.

## Goals
- simple setup for a friend
- persistent config separate from code
- local web UI
- manual update checks / update prompts
- reusable vote runner

## Run

```bash
cd product-1
npm install
npm start
```

Open:
- http://localhost:4311

## Data
Stored in:
- `product-1/data/config.json`
- `product-1/data/status.json`
- `product-1/data/l2reborn-cookies.json`
